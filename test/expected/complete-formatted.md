### [.parser](#L27)

Set a parser that can later be used to parse any given string.

* `name` **{String}**
* `arr` **{Object|Array}**: Object or array of replacement patterns to associate.
* `returns` **{Strings}**: to allow chaining

```js
strings.parser (name, replacements)
```
